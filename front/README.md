# Frontend React

L'application cliente ou frontend est écrite en HTML, CSS et JavaScript avec le
framework React.

## Prérequis

- Node.JS
- npm

## Installation des dépendances npm

En ligne de commande, placez-vous dans le dossier `front` puis saisissez

```
npm install
```

Cela va installer le framework React et les autres dépendances du projet.

## Démarrage de l'application

Toujours en ligne de commande, saisissez

```
npm start
```

Après un temps de chargement, un navigateur web s'ouvre et se connecte à
l'application.
